## Ambiente Virtual de Python

Ativar o ambiente virtual:
```shell
source venv/bin/activate
```

Desligar o ambiente virtual:
```shell
deactivate
```

## Multipass VMs

Para criar uma instância execute:
```
multipass launch --name <instance-name>
```

Para ver as VMs criadas e o seu estado execute:
```
multipass list
```

Para iniciar uma sessão na linha de comandos execute:
```
multipass shell <instance-name>
```

Para desligar uma VM execute:
```
multipass stop <instance-name>
```
